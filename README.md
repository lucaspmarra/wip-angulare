## Running locally

Clone the repo

```bash
  git clone https://github.com/lucaspmarra/wip-angulare.git
```

Enter the project directory

```bash
  cd wip-angulare
```

Install the dependencies

```bash
  yarn
```

Start the dev server

```bash
  yarn dev
```
